📚 Rappel de cours – Les angles

1. Qu’est-ce qu’un angle ?

Un angle est une ouverture formée par deux demi-droites qui partent du même point. Ce point commun s’appelle le sommet.

2. Les principaux types

Angle aigu : plus petit qu’un angle droit.
Angle droit : 90°.
Angle obtus : plus grand que 90° mais plus petit que 180°.
Angle plat : 180°.

3. Le rapporteur

Pour mesurer un angle :

  1. placer le centre du rapporteur sur le sommet ;
  2. aligner le 0 sur un côté de l’angle ;
  3. lire la graduation à l’endroit où passe l’autre côté.

4. Repères utiles

Un quart de tour correspond à 90°.
Un demi-tour correspond à 180°.
Un tour complet correspond à 360°.

📚 Rappel de cours – Le tableur en 6e

En classe, nous pouvons utiliser Microsoft Excel 2007. À la maison, tu peux aussi utiliser Microsoft Excel (version plus récente), LibreOffice Calc ou OpenOffice Calc. Le principe reste le même : on entre des données dans des cellules, puis on utilise des formules pour calculer automatiquement.

1. Qu’est-ce qu’un tableur ?

Un tableur est un logiciel qui permet :

  • de ranger des données dans un tableau ;
  • de faire des calculs automatiques ;
  • de trier les données ;
  • de comparer des résultats ;
  • d’organiser des informations clairement.

Exemples de logiciels : Excel, LibreOffice Calc, OpenOffice Calc.

2. Ligne, colonne, cellule

Dans un tableur :

  • les colonnes sont repérées par des lettres : A, B, C…
  • les lignes sont repérées par des nombres : 1, 2, 3…
  • une cellule est une case du tableau.

Exemples de cellules : A1, B3, D5

3. Saisir des données

On peut écrire dans une cellule :

  • un mot ;
  • un nombre ;
  • une date ;
  • une formule.

Exemple :

ABC
1ObjetAngle
2Figure 135
3Figure 290

4. Une formule commence par =

Pour que le tableur comprenne qu’il doit calculer, on commence toujours par : =

Exemples :

  • =B2+C2
  • =C2-B2
  • =10+5

5. Additionner deux cellules

Pour additionner deux cellules : =B2+C2

Exemple concret : si B2 = 20 et C2 = 35, alors =B2+C2 donne 55.

6. Additionner plusieurs cellules

On peut aussi additionner plusieurs cellules : =B2+C2+D2

Ou utiliser une formule plus pratique sur une plage de cellules : =SOMME(B2:D2)

Cette formule additionne toutes les cellules de B2 à D2.

7. Additionner une plage de cellules

Une plage de cellules est un groupe de cellules voisines.

Exemple : B2:D2 signifie : B2, C2 et D2.

Pour tout additionner : =SOMME(B2:D2)

8. Soustraire

Pour calculer une différence : =C2-B2

Exemple : pour savoir combien il manque à un angle de 20° pour faire 90° : =90-20

9. Calculer un angle automatiquement

Le tableur peut aider à calculer des angles.

Exemple : si un angle droit mesure 90° et qu’un angle connu mesure 25°, on peut calculer l’angle manquant avec : =90-B2

Si B2 = 25, alors le résultat est 65.

10. Calculer la somme de plusieurs angles

Pour additionner plusieurs angles : =SOMME(B2:D2)

Exemple : si trois angles mesurent 20°, 30° et 40°, leur somme est : =20+30+40 ou =SOMME(B2:D2)

11. Trier des données

Trier signifie ranger des données dans un ordre.

  • du plus petit au plus grand ;
  • du plus grand au plus petit ;
  • par ordre alphabétique.

Exemple : ranger des angles de 15°, 90°, 40°, 120° du plus petit au plus grand.

12. Comment trier selon le logiciel ?

Avec Excel 2007 :

  • sélectionner les cellules ;
  • aller dans l’onglet Données ;
  • cliquer sur Trier de A à Z ou Trier de Z à A.

Avec Excel plus récent :

  • sélectionner le tableau ;
  • aller dans Données ;
  • choisir Trier du plus petit au plus grand ou l’inverse.

Avec LibreOffice Calc / OpenOffice Calc :

  • sélectionner les cellules ;
  • cliquer sur Données ;
  • choisir Trier ;
  • indiquer l’ordre souhaité.

13. Organiser ses données correctement

Pour bien travailler dans un tableur :

  • mettre un titre clair sur chaque colonne ;
  • écrire une seule information par cellule ;
  • éviter les lignes vides dans un tableau ;
  • vérifier les unités : °, cm, €…

Exemple :

FigureAngle connuAngle recherché
135°=90-B2
2120°=180-B3

14. Les compétences numériques travaillées

En utilisant un tableur, tu apprends à :

  • saisir et organiser des données ;
  • faire un calcul automatisé ;
  • trier des informations ;
  • vérifier un résultat.

Cela correspond à des compétences de travail sur les données, utiles en mathématiques et dans les compétences numériques.

15. À retenir

  • Une cellule = une lettre + un nombre.
  • Une formule commence par =.
  • Pour additionner : =B2+C2 ou =SOMME(B2:D2).
  • Pour soustraire : =C2-B2.
  • Le tableur peut calculer automatiquement un angle manquant.
  • On peut aussi trier des données pour mieux les lire.

🧩 Astuce organisation

Dans un tableur, range toujours tes données avec un titre par colonne. Cela aide à mieux lire, trier et vérifier les résultats.

💡 Astuces pour l’élève
